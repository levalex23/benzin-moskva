# 🚗 БензинМосква — народная карта АЗС

Краудсорсинговый сайт для отслеживания наличия бензина на заправках Москвы в реальном времени.

## Что умеет

- **Карта Москвы** с 20+ заправками (Лукойл, Газпромнефть, Роснефть, Татнефть и др.)
- **4 статуса:** есть бензин / очередь / лимит / нет бензина
- **Народные отчёты** — любой пользователь нажимает 1 кнопку
- **Live-лента** — последние события бегут снизу
- **Добавление АЗС** — пользователь кликает на карту и добавляет заправку
- **Firebase синхронизация** — все видят обновления в реальном времени

## Быстрый старт

### 1. Ключ Яндекс.Карт (бесплатно)

1. Зайди на https://developer.tech.yandex.ru/
2. Войди через Яндекс-аккаунт
3. Создай проект → выбери **«JavaScript API и HTTP Геокодер»**
4. Получи `apikey`
5. Открой `index.html`, найди строку:
   ```
   apikey=ВСТАВЬ_СВОЙ_КЛЮЧ_ЗДЕСЬ
   ```
   и замени на свой ключ.

### 2. Firebase (бесплатно, plan Spark)

1. Зайди на https://console.firebase.google.com/
2. Создай проект (например, `benzin-moskva`)
3. Перейди в **Realtime Database** → создай базу (выбери регион `europe-west1`)
4. Установи правила доступа (для MVP — открытые):
   ```json
   {
     "rules": {
       ".read": true,
       ".write": true
     }
   }
   ```
   > Для продакшена добавь авторизацию!
5. В **Project Settings** → **Your apps** → добавь Web App
6. Скопируй `firebaseConfig` и вставь в `index.html` в блок:
   ```js
   const firebaseConfig = { ... }
   ```

### 3. Запуск

Просто открой `index.html` в браузере — или залей на любой хостинг:

- **GitHub Pages** (бесплатно): залей репозиторий → Settings → Pages
- **Vercel** (бесплатно): `npx vercel` в папке проекта
- **Netlify** (бесплатно): перетащи папку на netlify.com/drop
